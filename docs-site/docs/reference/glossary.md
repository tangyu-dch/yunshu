# 术语表

| 术语 | 说明 |
| --- | --- |
| DID | 商户对外公布的呼入号码 |
| CTI | 呼叫中心业务控制层 |
| ESL | FreeSWITCH Event Socket Library |
| Leg A | 先建立的一侧通道 |
| Leg B | 后建立的一侧通道 |
| CDR | Call Detail Record，通话记录 |
| ACW | After Call Work，话后整理 |
| UAS | SIP User Agent Server |
| UAC | SIP User Agent Client |
| RTPEngine | RTP 媒体代理 |

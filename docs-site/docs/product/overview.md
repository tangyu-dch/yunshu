# 产品概览

云枢声讯是一套面向企业客服、电销、智能 IVR 与 SaaS 多租户场景的呼叫中心平台。

## 产品能力

```text
客户触达
  ├─ 云枢声讯 坐席云枢声讯
  ├─ API 外呼
  ├─ 批量外呼
  └─ 客户呼入 DID

智能服务
  ├─ AI IVR
  ├─ ASR/TTS/LLM
  └─ RAG 知识库

运营闭环
  ├─ CDR 通话记录
  ├─ 计费流水
  ├─ 录音归档
  ├─ 报表投影
  └─ Webhook 下游推送
```

## 核心价值

- 用 Go 重构传统呼叫中心核心链路，降低运行时复杂度。
- 使用事件和工作流表达通话生命周期，便于测试和排障。
- 将通话记录、计费、录音、下游推送拆成可靠 outbox 节点。
- 兼容 SIP 云枢声讯、FreeSWITCH、Kamailio、RTPEngine 的生产级部署。

## 典型场景

| 场景 | 描述 |
| --- | --- |
| 客服中心 | 客户拨打 DID，系统按技能组分配坐席 |
| 电销系统 | API 或批量任务主动外呼客户 |
| 智能 IVR | 机器人接待、意图识别、转人工 |
| SaaS 多商户 | 多商户隔离管理分机、号码池、网关与账务 |
